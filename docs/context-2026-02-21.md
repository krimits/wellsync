### Πιθανές Ερωτήσεις Q&A

**Τεχνικές:**
- *"Γιατί Ridge Regression και όχι neural network;"*
  → Interpretable, lightweight, αρκεί για demo
- *"Τι γίνεται αν το Claude API δεν απαντήσει;"*
  → ML-only fallback — πάντα δουλεύει
- *"Πώς προστατεύετε τα δεδομένα υγείας;"*
  → JWT auth, local PostgreSQL, μόνο aggregated metrics στο Claude API
- *"Γιατί APScheduler και όχι Celery;"*
  → Zero extra Docker service, αρκεί για single-instance demo

**Ιδέα / Αγορά:**
- *"Πώς διαφέρετε από το Apple Health;"*
  → Apple Health συλλέγει, ΔΕΝ παράγει personalized AI σύσταση σε φυσική γλώσσα
- *"Ποιο είναι το business model;"*
  → Freemium: βασικά δωρεάν, premium για advanced ML insights
- *"14 μέρες cold start — πώς κρατάς τον χρήστη;"*
  → Rule-based readiness από μέρα 1 (άμεση αξία), ML ενεργοποιείται αθόρυβα
